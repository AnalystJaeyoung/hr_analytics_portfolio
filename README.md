# HR Analytics Portfolio

## Project 1 · Turnover Prediction Model
**Goal:** Predict employee attrition using Logistic Regression

**Dataset:** IBM HR Analytics Dataset (1,470 employees, 35 variables)

**Key Results:**
- Best model: Logistic Regression with class balancing
- Recall: 0.69 (captures 69% of actual attrition cases)
- Key predictors: OverTime, JobSatisfaction, MonthlyIncome

**Tech Stack:** Python, pandas, scikit-learn, seaborn, statsmodels

## Skills Demonstrated
- Exploratory Data Analysis (EDA)
- Class Imbalance handling
- Model comparison (Logistic Regression vs Random Forest)
- Statistical interpretation (Odds Ratio, p-value)
